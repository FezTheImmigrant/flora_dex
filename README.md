# Flora Dex
## Description
- This project is aimed at classifying a number of wildflowers native to Colorado.
## Prerequisites
- docker
- docker-compose >= 1.28
- nvidia-container-runtime
### Model Description
- Resnet50
  - Input Layer: 256,256,3
  - Total params: 157,890,514
  - Activation Function: softmax
